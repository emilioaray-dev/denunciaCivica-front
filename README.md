# Denuncia Civica – Frontend

Frontend de la plataforma Denuncia Civica. Next.js 15 con App Router, Tailwind CSS, shadcn/ui.

## Stack

- Next.js 15 (App Router)
- TypeScript (strict)
- Tailwind CSS 4 + shadcn/ui
- React Hook Form + Zod
- TanStack Query

## Spec Kit

La documentacion completa del proyecto se encuentra en `specs/mvp-denuncia-civica/`:

- `spec.md` – PRD, user stories, requirements
- `constitution.md` – Principios y restricciones del frontend
- `plan.md` – Plan de implementacion y estructura
- `tasks.md` – Backlog de tareas de ingenieria
- `research.md` – Analisis del design system de referencia

## Ramas

- `main` – Produccion
- `preview` – Iteracion Vercel
- `feat/*`, `fix/*`, `chore/*` – Feature branches (conventional commits)
